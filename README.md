# DataTranquil Company Website

Modern company website for DataTranquil - AI-Powered Data Solutions

## Features

- **Modern Design**: Inspired by BrightData with clean, professional aesthetics
- **Product Showcase**: Featuring MoneyTrail and upcoming products
- **Service Grid**: AI Development, Data Analytics, Consulting, Automation
- **Responsive Layout**: Mobile-first design with Tailwind CSS
- **Interactive Elements**: Hover effects, gradients, and smooth transitions
- **SEO Optimized**: Proper meta tags and structured content

## Tech Stack

- **Next.js 14.2.5** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Lucide React** for modern icons

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy to Vercel with:
- Framework: Next.js
- Build Command: `npm run build`
- Output Directory: `.next`

## Domain Setup

This website is designed to be deployed at `datatranquil.com` with:
- MoneyTrail app at `moneytrail.datatranquil.com`
- Company website at root domain

## Project Structure

```
src/
```
