# Git Workflow Commands

## /smart-commit
Intelligent git commit with automatic enhancements.
- Groups related changes
- Generates semantic commit messages (feat, fix, refactor)
- Runs pre-commit checks
- Includes context and references

## /batch-ops
Execute operations across multiple files efficiently.
- Rename functions/variables everywhere
- Update API patterns
- Add headers/imports to multiple files
- Bulk create tests/types/docs

## Quick Git Shortcuts
- `undo-last` - Undo last commit (keep changes)
- `clean-branches` - Delete merged branches  
- `sync-fork` - Sync with upstream
- `cherry-pick-pr` - Cherry-pick from PR
