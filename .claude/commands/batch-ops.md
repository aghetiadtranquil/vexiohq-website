# Batch Operations

Execute common operations across multiple files efficiently.

## Operations:
1. **Rename everywhere**
   - Function/variable/class names
   - Updates imports, exports, references

2. **Add to all files matching pattern**
   - License headers
   - Import statements
   - Logger initialization

3. **Update patterns**
   - Old API → New API
   - Callback → Async/await
   - Class components → Functional

4. **Bulk create**
   - Tests for all components
   - Types for all API endpoints
   - Documentation for all functions

## Example:
"Rename getUserData to fetchUserProfile everywhere"
"Add logger to all service files"
"Convert all .js files to .ts with proper types"

Saves hours of repetitive work!
