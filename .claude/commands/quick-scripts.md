# Quick Scripts

Common one-liners and scripts for instant execution.

## Database
- `db-backup`: Backup database with timestamp
- `db-restore`: Restore from latest backup
- `db-migrate`: Run pending migrations
- `db-seed`: Seed development data

## Development
- `clean-all`: Remove node_modules, cache, build
- `fresh-start`: Clean + install + build
- `update-deps`: Update all dependencies safely
- `security-check`: Run security audit

## Debugging
- `find-todos`: List all TODO/FIXME comments
- `unused-code`: Find unused exports/functions
- `large-files`: Find files over 500 lines
- `complexity`: Find complex functions

## Performance
- `bundle-analyze`: Analyze bundle size
- `lighthouse`: Run Lighthouse audit
- `profile`: Start profiling mode
- `find-n+1`: Detect N+1 queries

## Git Shortcuts
- `undo-last`: Undo last commit (keep changes)
- `clean-branches`: Delete merged branches
- `sync-fork`: Sync with upstream
- `cherry-pick-pr`: Cherry-pick from PR

Just type the shortcut name to run!
