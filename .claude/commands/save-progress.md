# Save Progress

Update SESSION.md with current work state for next session.

## What to Save:
1. Current task and progress
2. Recent changes made
3. Pending tasks from TodoWrite
4. Any errors or blockers
5. Next logical steps
6. Important context or decisions made

## Also Update:
- TODO.md with remaining tasks
- CLAUDE.md with any new learnings

This ensures smooth handoff between sessions.
